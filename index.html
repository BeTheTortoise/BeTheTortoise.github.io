Fantasy Football Scheduler

In the program, I prompt the user for basic input on the structure of their league: number of teams, number of divisions, and number of playoff teams. Then, I create a dictionary to hold the entire schedule and randomize the divisions. Also, I make a list of every possible game to use a reference when assigning games later to make sure it hasn't been assigned already. As I assign games, I create new lists of possible remaining games which is limited by home/away game counts and divisional game requirements. The assignment uses recursion, assigning game after game until either the schedule is complete or no games are possible to be assigned.
I'm currently working on saving the schedules as they are made so that the program doesn't make the same schedule twice. Check out the source code on my GitHub. A sample schedule is below.
